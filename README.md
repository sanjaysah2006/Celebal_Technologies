# Celebal Technologies Repository

This repository contains weekly data analysis and SQL assignments for the Celebal Technologies training program.

## Repository Contents

### 📁 week1/
Week 1 datasets and notebooks for the initial training tasks.

### 📁 week2/
Week 2 SQL database design and Superstore analysis assignments.

### 📁 week3/
Week 3 Superstore dataset analysis, SQL scripts, and Jupyter notebooks.

## Learning Outcomes

* SQL database design and query writing
* Data loading and preprocessing
* Exploratory data analysis in Jupyter Notebook
* Aggregations, joins, and performance considerations
* Business insights from sales datasets
