# Sales Data Analysis & SQL Database Design

This repository contains Week 2 tasks focused on SQL database design, data loading, querying, and sales data analysis using MySQL and Jupyter Notebook.

## Repository Contents

### 📄 CEI_SQL_TASK2_WEEK2.sql

A complete SQL database implementation for an e-commerce sales system.

#### Features

* Database creation and selection
* Table creation with constraints
* Primary and Foreign Keys
* Data validation using CHECK constraints
* Index creation for performance optimization
* Sample data insertion
* SQL queries for data analysis and reporting

#### Database Structure

* **customers**
* **products**
* **orders**
* **order_items**

#### Concepts Covered

* DDL (CREATE, ALTER)
* DML (INSERT, UPDATE)
* Constraints
* Indexing
* Joins
* Aggregations
* Data Analysis Queries
